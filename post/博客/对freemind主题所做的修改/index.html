2014-08-16
博客搭建

文章目录

  • 现已改用Jackman主题（做了些许修改）*

修改分类方式

原主题的侧边样的分类是无层次的分类，在此改为层次型的分类方式，好在hexo原本就支持这种分类方式。修改theme/freemind/layout/_widget/category.ejs文件如下：

3
4
5

7

<% if (site.categories.length){ %>
	<div class="widget">
		<h4><%= __('categories') %></h4>
		<ul class="tag_box list-unstyled">
			<%- list_categories() %>
		</ul>
	</div>
<% } %>
